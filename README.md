This is a starter template for a React Native project with Expo Router and Native Wind installed.

Versions :

- React Native 0.81.5
- Expo 54
- Native Wind 4.2.1

## What you get?

- A starter kit for a recta native project with tabs
- Using expo router and native tabs (liquid glass) - Make sure to run with iOS 26
- TailwindCSS ready to rock

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Open app.json and update expo>name, expo>slug, expo>scheme and update the icon and splash if needed.
3. Open Package.json and change the name accordingly to you project
4. Start the app

   ```bash
   npx expo start
   ```
